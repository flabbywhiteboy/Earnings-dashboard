# Stock Watchlist Pro

This version adds:
- your fuller holdings list
- live prices
- next earnings dates
- a browser-saved Finnhub API key
- mobile-friendly layout
- GitHub Pages compatibility

## How to use
1. Upload these files to your GitHub repo, replacing the older frontend files.
2. Wait for GitHub Pages to redeploy.
3. Open the live site.
4. Get a Finnhub API key.
5. Paste the API key into the app and tap **Save key**.
6. Tap **Refresh**.
